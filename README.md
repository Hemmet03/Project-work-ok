# Guida Sicurezza sul Lavoro: Valutazione Rischi 81/08 e ISO 31000

Una Landing Page professionale progettata per offrire una guida completa sulla valutazione dei rischi nei luoghi di lavoro. Il progetto si rivolge a Datori di Lavoro, RSPP e consulenti della sicurezza, integrando normative italiane (D.Lgs 81/08) e standard internazionali (ISO 31000).

[Visualizza la Demo Live](https://hemmet03.github.io/Project-work-ok/)

## Caratteristiche
- **Design Responsive:** Ottimizzato per Desktop, Tablet e Mobile grazie a Bootstrap 5.
- **Generazione QR Code:** Script integrato per generare dinamicamente un QR Code che punta al documento PDF.
- **Download Risorse:** Sezione dedicata al download diretto della guida tecnica.
- **SEO Ready:** Meta-tag ottimizzati per indicizzare i contenuti relativi alla sicurezza sul lavoro.
- **Accessibilità:** Uso di icone intuitive e contrasti cromatici studiati per la leggibilità.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Styles + Bootstrap 5.3.3)
* **Icone:** Bootstrap Icons
* **Tipografia:** Google Fonts (Inter)
* **Scripting:** JavaScript (per la generazione del QR Code)

## Librerie utilizzate
Il progetto sfrutta le seguenti risorse esterne:
* **Bootstrap 5**: Framework CSS.
* **QRCode.js**: Libreria JavaScript per la generazione di codici QR.
* **Google Fonts**: Font "Inter".
